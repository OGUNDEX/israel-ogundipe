The Weather Digital

Role: Technical Consultant

Problem

  • Mobile PageSpeed score in the low 30s
  • Poor INP due to heavy frontend scripts
  • Hero media blocking LCP

Diagnosis

  • Excessive third-party scripts
  • Render-blocking hero assets
  • Unoptimized asset delivery path

Solution

  • Proposed Cloudflare R2 media decoupling
  • Reduced render-blocking assets above the fold
  • Optimized script execution order

Impact

  • Projected 1.2s LCP reduction
  • Significant INP improvement roadmap

ElimuVillage

Role: Lead Developer

Problem

  • Multiple YouTube embeds causing heavy script bloat
  • Unstable layout shifts (CLS)
  • Slow mobile experience

Diagnosis

  • Third-party embed overhead
  • Unoptimized media loading strategy

Solution

  • Replaced YouTube embeds with lightweight alternatives
  • Self-hosted media via Cloudflare R2
  • Improved loading strategy for above-the-fold assets

Impact

  • 11% immediate performance improvement
  • CLS stabilized across mobile and desktop
